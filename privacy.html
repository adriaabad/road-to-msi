SUMMARY

What this site collects.

Road to MSI is a static browser game. It does not require an account, does not ask for personal information, and does not use advertising trackers.

  • No cookies are set by the game.
  • No login, payment, newsletter, or contact form is provided by the game.
  • Game data is loaded from a local JSON file so the app can run in the browser.
  • If visit counting is enabled, it should use a privacy-friendly analytics provider without cookies, such as Cloudflare Web Analytics.
  • External links to LinkedIn and Discord are governed by those services' own privacy policies.
